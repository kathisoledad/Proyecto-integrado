Diseñador Web:

Es el encargado de dibujar la página web que luego va a construir el desarrollador.

Desarrollador Web:

Un desarrollador web es el encargado de programar páginas que luego se van a ejecutar en la world wide web mediante varios tipos de lenguajes.

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico

Hojas de estilo en cascada es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado

Bootstrap es un framework o conjunto de herramientas de Código abierto para diseño de sitios y aplicaciones web.

HTML, sigla en inglés de HyperText Markup Language, hace referencia al lenguaje de marcado para la elaboración de páginas web.
